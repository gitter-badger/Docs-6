========================
MinigameCore Development
========================

Overview
========

MinigameCore is a large project, and help from any developers is greatly appreciated. 
This page describes how to contribute to the main MinigameCore project.

.. warning::

	You should have prior experience with Java and the SpongeAPI before 
	contributing to the main MinigameCore project.
	
Getting Started
===============

There are a few steps to follow when contributing to MinigameCore:

#. Fork MinigameCore on GitHub

#. Clone the forked repository locally onto your computer

#. Import the project into your IDE (Be sure to import it as a Maven project)

#. Make your desired changes to MinigameCore

#. Commit the changes to your repository

#. Open a pull request with your changes
