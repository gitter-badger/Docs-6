MinigameCore Development¶

Overview¶

MinigameCore is a large project, and help from any developers is greatly appreciated. This page describes how to contribute to the main MinigameCore project.

Warning

You should have prior experience with Java and the SpongeAPI before contributing to the main MinigameCore project.

Getting Started¶

There are a few steps to follow when contributing to MinigameCore:

  1. Fork MinigameCore on GitHub
  2. Clone the forked repository locally onto your computer
  3. Import the project into your IDE (Be sure to import it as a Maven project)
  4. Make your desired changes to MinigameCore
  5. Commit the changes to your repository
  6. Open a pull request with your changes
