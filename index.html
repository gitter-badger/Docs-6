MinigameCore Documentation¶

Introduction¶

Welcome to MinigameCoreDocs, the official documentation for MinigameCore.

Warning

MinigameCore is still in development, so some functions are not complete or not implemented. Also note that portions of MinigameCore aren’t tested yet, so expect some bugs to arise. If you find a bug, please immediately report it to the GitHub issue tracker.
